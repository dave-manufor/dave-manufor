## 🚀 What I'm Currently Building
- 🎯 **[ViralCue](https://github.com/dave-manufor/viralcue)** — A real-time AI co-pilot for creators. It detects viral moments from live audio streams on the fly using Deepgram and AWS Bedrock. Built as a Turborepo monorepo with Next.js, Express, and Python.
- ⚖️ **[Clausync AI](https://github.com/dave-manufor/clausync-ai)** — An AI-powered contract analysis platform. Uses a distributed worker architecture for ingestion, LLM analysis, and vectorization (RAG) to help teams manage legal documents at scale.
- 🚑 **[Responder AI](https://github.com/dave-manufor/responder-ai)** — An AI-powered emergency response assistant leveraging Google's Agent Development Kit (ADK) for intelligent triage and geospatial hospital routing.
- 🌉 **[BlackBridge](https://github.com/dave-manufor/blackbridge-app)** — A full-stack TypeScript application with Terraform-managed AWS infrastructure and automated CI/CD pipelines.
